# react_kadai02

## ①プロダクトの紹介
- 音声を入力したら画面上に自動的に文字として出力されるアプリです！
- 「Web Speech API」という、ブラウザの音声認識機能を使うためのAPIを使用しました
- ※Google Chromeブラウザでしか動作しません

## ②工夫した点
- 日本語を認識して日本語の文字が出力されるようにしたところ
